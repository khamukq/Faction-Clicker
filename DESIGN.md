## 🎯 КОНЦЕПЦИЯ ИГРЫ

### Краткое описание

**Faction Clicker** — это инкрементальная RPG с глубокими механиками прогресса и социальным взаимодействием. Игрок выбирает одну из трёх фракций, сражается с врагами, прокачивает персонажа и соревнуется с другими игроками в таблице лидеров.

### Ключевые особенности

- **Выбор фракции** — 3 уникальные фракции с разными стилями игры
- **Глубокий прогресс** — уровни, этажи, престиж, вознесение
- **Активные способности** — уникальные скиллы каждой фракции
- **Социальные механики** — кланы и лидерборд
- **Автокликер** — автоматическая прокачка
- **Система комбо** — бонусы за непрерывные атаки

### Целевая аудитория

- Любители инкрементальных игр (clicker/idle)
- Игроки, предпочитающие стратегическое планирование
- Социальные игроки (кланы, соревнование)
- Возраст: 14+

### Основной цикл геймплея

```
Клик по врагу → Получение золота и XP → Улучшение персонажа → 
Убийство босса → Переход на следующий этаж → 
Престиж (сброс ради множителя) → Вознесение (следующий уровень)
```

## ⚔️ ОСНОВНЫЕ МЕХАНИКИ

### 1. Кликер (Основное действие)

- **Что делает:** Игрок кликает по врагу, нанося урон
- **Частота:** Каждый клик = одна атака
- **Урон:** Зависит от уровня, улучшений, фракции, престижа, вознесения
- **Анимация:** Визуальный отклик, всплывающие цифры урона
- **Комбо-система:** Серия атак увеличивает урон (до +15%)

### 2. Автокликер

- **Что делает:** Автоматически атакует врагов с заданным интервалом
- **Как активировать:** Купить за 100 золота
- **Улучшение:** Увеличение скорости (макс. уровень 10)
- **Интервал:** Начинается с 1000мс, уменьшается на 50мс за уровень
- **Мин. интервал:** 100мс

### 3. Армия (Наёмники)

- **Что делает:** Даёт пассивные бонусы к урону и доходу
- **Стоимость:** Растёт экспоненциально
- **Эффект:** +2 урона + бонусы фракции за каждого наёмника
- **Максимум:** Не ограничен

### 4. Боссы

- **Частота:** Каждые 25 убийств
- **Сложность:** В 2.5-5 раз сильнее обычных врагов
- **Награда:** Больше золота и XP
- **Механика:** 5 попыток на победу, затем можно пропустить за 200 золота

### 5. Супер-боссы

- **Частота:** Каждые 500 этажей
- **Сложность:** В 10+ раз сильнее обычных боссов
- **Награда:** x2 золота и XP от обычного босса
- **Особенность:** Не атакуют в ответ

### 6. Комбо-система

- **Эффект:** +0.2% урона за удар в комбо
- **Макс. бонус:** +15%
- **Сброс:** Через 3 секунды без атак

### 7. Критические удары

- **Базовый шанс:** 0%
- **Макс. шанс:** 70%
- **Базовый множитель:** x1.5
- **Улучшения:** +2% за уровень улучшения "Меткость"

## 📈 ПРОГРЕСС ИГРОКА

### Уровни (Level)

```
Максимальный уровень: 100
Формула XP: 50 * 1.25^(level-1)
Бонусы за уровень:
  - Урон: +1.5 за уровень
  - HP: +8 за уровень
  - Регенерация: +0.2/сек за уровень
  - Доход: +0.5/сек за уровень
```

#### Вехи (Milestones):

| Уровень | Награда | Звание |
|---|---|---|
| 5 | 50 золота | Новичок |
| 10 | 100 золота | Воин |
| 25 | 300 золота | Ветеран |
| 50 | 800 золота | Герой |
| 75 | 1500 золота | Легенда |
| 100 | 3000 золота | Миф |

### Этажи (Floors)

```
Врагов на этаж: 50
Сложность: Растёт на 10% за этаж
Награда: Бонус к золоту и XP
Супер-босс: Каждые 500 этажей
```

### Престиж (Prestige)

```
Требование: 100,000+ золота
Формула очков: floor(log(1 + gold/100000)^1.1)
Эффект: Сброс прогресса → постоянный множитель
Макс. множитель: x3.0
Сохраняется: Перки
```

### Вознесение (Ascension)

```
Требование: 5+ очков престижа
Формула очков: floor(log(1 + prestigePoints) * 0.5)
Эффект: Сброс престижа → +10% к силе за уровень
Сохраняется: Перки, достижения
```

## 💰 ЭКОНОМИКА

### Источники золота

1. **Убийства врагов** — основной источник
1. **Пассивный доход** — каждую секунду
1. **Бонусы фракций** — пассивные способности
1. **Достижения** — разовые награды
1. **Боссы** — крупные награды
1. **Супер-боссы** — двойная награда

### Расход золота

1. **Улучшения** — прокачка характеристик
1. **Найм армии** — постоянные бонусы
1. **Автокликер** — активация и улучшение
1. **Создание клана** — 5000 золота
1. **Пропуск босса** — 200 золота

### Формулы

```
// Урон
baseDamage = 4 + level * 0.8
armyDamage = armyCount * 2
totalDamage = (baseDamage + levelBonus + armyDamage + bonuses) 
    * factionMult * prestigeMult * ascensionMult

// Доход
passiveIncome = (passiveBonuses + goldBonuses + armyCount * armyPassiveBonus) 
    * factionMult * prestigeMult * ascensionMult

// Стоимость улучшений
cost = baseCost * (1.3 ^ currentLevel)

// Стоимость армии
hireCost = 120 * (1.2 ^ armyCount)
```
